# 사주팔자 웹사이트

한국 전통 사주팔자 해석 웹사이트입니다.

## 🔮 주요 기능

### 1. 사주팔자 해석
- 생년월일시 입력으로 정확한 사주 계산
- 일간 분석 및 성격 해석
- 십성 분석 및 오행 균형 검토
- 대운 분석 (10년 단위 운세)
- 올해 및 월별 운세

### 2. 오늘의 운세
- 일반적인 오늘의 운세 제공
- 행운의 색상, 방향, 시간 안내
- 주의사항 및 조언

### 3. 궁합 분석
- 두 사람의 사주 비교 분석
- 연애, 결혼, 사업, 우정 궁합 점수
- 상세한 궁합 해석 및 조언
- 일간 상성 및 오행 균형 분석

### 4. 택일 (개발 예정)
- 사주 기반 좋은 날 찾기
- 결혼, 이사, 개업 등 목적별 길일 추천

## 🛠️ 기술 스택

- **Frontend**: Next.js 14, React 18, TypeScript
- **Styling**: Tailwind CSS
- **Animation**: Framer Motion
- **Icons**: Heroicons

## 🚀 설치 및 실행

### 1. 의존성 설치
```bash
npm install
```

### 2. 개발 서버 실행
```bash
npm run dev
```

### 3. 프로덕션 빌드
```bash
npm run build
npm start
```

## 📖 사주 계산 로직

### 간지 계산
- **년주**: 1984년을 기준으로 한 간지 계산
- **월주**: 년간에 따른 월간 계산
- **일주**: 2000년 1월 1일을 기준으로 한 일수 계산
- **시주**: 일간과 시간에 따른 시간 계산

### 오행 분석
- 목(木), 화(火), 토(土), 금(金), 수(水)의 개수 계산
- 상생상극 관계 분석
- 오행 균형 및 부족한 요소 분석

### 십성 분석
- 일간을 기준으로 한 십성 계산
- 비견, 겁재, 식신, 상관, 편재, 정재, 편관, 정관, 편인, 정인
- 십성별 성격 및 운세 해석

### 대운 계산
- 성별과 년간의 음양에 따른 대운 방향 결정
- 10년 단위 대운 간지 계산
- 대운별 해석 제공

## 🎨 디자인 특징

### 동양적 디자인
- 한국 전통 색상 (금색, 적색, 청색 등)
- 한자 폰트 활용
- 그라데이션 배경과 글래스모피즘 효과

### 반응형 디자인
- 모바일, 태블릿, 데스크톱 대응
- Tailwind CSS를 활용한 반응형 레이아웃

### 사용자 경험
- 부드러운 애니메이션 (Framer Motion)
- 직관적인 네비게이션
- 단계별 안내 시스템

## 📊 주요 컴포넌트

### SajuForm
- 생년월일시 입력 폼
- 유효성 검사 및 에러 처리
- 성별 선택 및 시간 선택

### SajuResult
- 사주팔자 표 시각화
- 탭 기반 결과 표시
- 오행 분석 차트

### Navigation
- 전체 페이지 네비게이션
- 현재 페이지 표시
- 반응형 메뉴

## 🔍 주요 기능 상세

### 사주 해석
1. **기본 해석**: 일간 분석, 십성 분석
2. **성격 분석**: 오행 기반 성격 분석
3. **운세**: 로또 번호, 일반 운세
4. **대운**: 10년 단위 대운 해석
5. **올해 운세**: 년 운세 및 월별 상세 운세

### 궁합 분석
1. **기본 정보**: 두 사람의 사주 비교
2. **궁합 점수**: 연애, 결혼, 사업, 우정별 점수
3. **상세 해석**: 일간 상성, 오행 균형 분석
4. **조언**: 관계 발전을 위한 구체적 조언
